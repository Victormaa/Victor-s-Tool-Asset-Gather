# ESP32与Unity通过局域网通信的注意事项

## 硬件要求
- **ESP32模块**（价格约10+元，性价比高）
- **2.4GHz WiFi网络**（ESP32不支持5GHz频段）

## 三大常见问题与解决方案

### 1. 网络频段问题
- 🚨 **必须使用2.4GHz网络**（硬件限制）
- 5GHz网络会导致连接失败

### 2. 端口配置
- 🔍 **必须验证端口可用性**：
  ```arduino
  // Arduino示例代码
  if(!server.begin(8080)) { 
    Serial.println("端口占用！");
  }
  ❌ 不要完全依赖AI推荐的端口

✅ 建议使用49152-65535范围内的动态端口
### 3. 防火墙设置
- 🛡️ 需要配置的规则：

- 在防火墙设置允许UDP/TCP出入站连接的规则

- 关闭公共网络防火墙（测试时）
